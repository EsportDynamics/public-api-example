Happy Path Checklist

1. Configure Set RANKACY_BASE_URL and RANKACY_TOKEN.
2. Upload Send a .dem file, with optional auto highlight settings.
3. Poll Watch demo status until it reaches SUCCESS or FAILED.
4. Inspect Use kills and players to decide what to render.
5. Create Queue a highlight automatically, by ticks, or by kill ID from the Demos page.
6. Monitor Track highlight status and webhook deliveries.
